# Kairos.Artifex Portfolio

A highly premium Next.js digital product studio portfolio designed with pixel-perfect Figma canvas interactions.

## Features

- **Figma Canvas Sandbox**: Draggable and selectable design components (Design, Develop, Launch, and Subheading) mapping coordinate space dynamically.
- **Micro-interactions**: Instant outline selections matching Figma behavior with standard Tailwind v4 border radii.
- **GPU-Accelerated Rendering**: Smooth dragging mechanics powered by GSAP with zero subpixel font anti-aliasing jitter.
- **Custom Assets**: Includes customized SVG cursors for canvas navigation.
- **Design System**: Typography specs strictly built around the Inter font face, #F7F7F7 background, and a prominent dotted background grid.

## Getting Started

First, run the development server:

```bash
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
